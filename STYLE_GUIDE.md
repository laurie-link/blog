# 博客样式系统说明

本博客采用了完整的 Markdown 渲染系统，能够自动将 Markdown 文本转换为精美的 HTML 样式。

## 🎨 支持的样式元素

### 1. 标题 (H1-H6)

```markdown
# 一级标题 - 带下边框
## 二级标题 - 带下边框
### 三级标题
#### 四级标题
```

**样式特点**：
- H1：4xl 大小，主色渐变，粗下边框
- H2：3xl 大小，灰色细下边框
- H3-H6：递减大小，保持层次清晰

### 2. 文本样式

```markdown
**粗体文本**
*斜体文本*
***粗斜体***
~~删除线~~
```

**样式特点**：
- 粗体：深色加粗
- 斜体：倾斜显示
- 删除线：灰色中划线

### 3. 链接

```markdown
[链接文本](https://example.com)
```

**样式特点**：
- 主题色显示
- 下划线装饰
- 悬停时颜色加深
- 自动打开新标签页

### 4. 列表

```markdown
- 无序列表项
  - 嵌套列表

1. 有序列表项
   1. 嵌套列表
```

**样式特点**：
- 圆点/数字使用主题色
- 适当的缩进和间距
- 支持多层嵌套

### 5. 代码

#### 行内代码

```markdown
这是 `行内代码` 示例
```

**样式特点**：
- 浅色背景
- 主题色文字
- 边框装饰
- 等宽字体

#### 代码块

````markdown
```javascript
function hello() {
  console.log("Hello World")
}
```
````

**样式特点**：
- 深色背景（暗黑主题）
- 语法高亮（支持 180+ 语言）
- 自定义滚动条
- 圆角阴影效果

**支持的语言**：JavaScript, TypeScript, Python, Java, CSS, HTML, Bash, JSON, SQL, Go, Rust 等

### 6. 引用

```markdown
> 这是引用文本
> 可以跨多行
```

**样式特点**：
- 左侧主题色边框
- 淡色背景
- 斜体显示
- 圆角效果

### 7. 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|:---:|----:|
| 左对齐 | 居中 | 右对齐 |
```

**样式特点**：
- 斑马纹背景
- 悬停高亮
- 边框装饰
- 响应式横向滚动

### 8. 图片

```markdown
![图片描述](图片URL)
```

**样式特点**：
- 自动居中
- 圆角边框
- 阴影效果
- 懒加载优化
- 响应式适配

### 9. 分隔线

```markdown
---
***
___
```

**样式特点**：
- 2px 粗细
- 灰色显示
- 上下间距

### 10. 任务列表

```markdown
- [x] 已完成
- [ ] 待完成
```

**样式特点**：
- 复选框样式
- 清晰的状态显示

## 📝 使用示例

创建新文章时，使用以下格式：

```markdown
---
title: 文章标题
date: 2024-01-15
excerpt: 文章摘要
tags: [标签1, 标签2]
---

# 文章标题

文章正文...

## 章节标题

内容...
```

## 🎨 样式定制

### 主题色修改

在 `tailwind.config.ts` 中修改：

```typescript
colors: {
  primary: {
    500: '#0ea5e9', // 修改为你喜欢的颜色
    // ...
  }
}
```

### 代码高亮主题

在 `app/globals.css` 中修改：

```css
@import 'highlight.js/styles/github-dark.css'; /* 更换为其他主题 */
```

可选主题：
- `github-dark.css` - GitHub 暗色（默认）
- `atom-one-dark.css` - Atom 风格
- `vs2015.css` - VS Code 暗色
- `monokai.css` - Monokai

### 自定义样式

在 `app/globals.css` 中添加自定义样式。

## 🔧 技术实现

- **渲染引擎**: react-markdown
- **GFM 支持**: remark-gfm
- **代码高亮**: rehype-highlight + highlight.js
- **HTML 支持**: rehype-raw
- **样式框架**: Tailwind CSS
