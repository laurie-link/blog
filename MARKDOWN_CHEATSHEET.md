# Markdown 快速参考

## 基础语法

### 标题
```markdown
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

### 文本样式
```markdown
**粗体**
*斜体*
***粗斜体***
~~删除线~~
```

### 链接和图片
```markdown
[链接文字](https://example.com)
![图片描述](image.jpg)
```

### 列表
```markdown
- 无序列表项
- 另一项
  - 嵌套项

1. 有序列表项
2. 另一项
   1. 嵌套项
```

### 代码
```markdown
`行内代码`

```javascript
// 代码块
function hello() {
  console.log("Hello")
}
```
```

### 引用
```markdown
> 引用文本
> 可以跨多行
```

### 分隔线
```markdown
---
***
___
```

## GFM 扩展

### 表格
```markdown
| 列1 | 列2 | 列3 |
|-----|:---:|----:|
| 左 | 中 | 右 |
```

### 任务列表
```markdown
- [x] 已完成
- [ ] 待完成
```

### 删除线
```markdown
~~删除的文本~~
```

## 快速提示

1. **标题前后留空行** - 提高可读性
2. **代码块指定语言** - 获得语法高亮
3. **链接添加描述** - 提供更多上下文
4. **表格对齐** - 使用冒号控制对齐
5. **图片优化** - 使用适当大小的图片
